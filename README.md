## 👏说明

本仓库是对 STEAM 爆款游戏 [Helltaker](https://store.steampowered.com/app/1289310/Helltaker/) 中文汉化项目，主要负责为大家提供免费优质的中文汉化包。

在查阅了大量资料并借助翻译工具的辅助，本人最终制作出了 v1.0 版本的汉化包。但是由于本人翻译水平的有限，本人只能尽自己最大努力保证语境的正确。虽然本人也想翻译得“信”、“达”、“雅”，但是在翻译过程中确实有很多地方拿不准。这些句子既不能直译，还需要还原英文语境的意思，这让我感到十分棘手。所以，如果大家对本项目中任何地方有疑问或者建议，都可以在 [Issues](https://github.com/SeaEpoch/Helltaker-Chinese/issues) 中提出你的想法。

## ✅如何安装？

这非常的简单，你只需要在 Steam 中找到游戏，然后浏览游戏根目录，最后将项目中的 local 文件夹与 localHM 文件夹替换游戏根目录中的同名文件夹及文件夹中文件即可。
